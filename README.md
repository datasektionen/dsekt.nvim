# dsekt.nvim
[Konglig Datasektionen](https://datasektionen.se/)'s official neovim colorscheme, designed by Data's Art Director.

![Alt text](public/image.png)

### Installation

Using `lazy`

```
{
    "datasektionen/dsekt.nvim",
    priority = 1000,
}
```
